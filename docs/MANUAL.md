# 📖 テロップエディター 初心者向けマニュアル

---

## このマニュアルでわかること

- アプリの起動方法
- 各ボタンの意味
- テロップの編集方法
- 動画に反映させるまでの手順

---

## 🟦 Step 1：アプリを起動する

1. **ターミナル**（Mac に最初から入っているアプリ）を開く
   - Launchpad（ロケットのアイコン）→「ターミナル」で検索
2. **解凍したフォルダをターミナルの黒い画面にドラッグ＆ドロップ**する

> 💡 ドラッグすると `cd /Users/あなたの名前/...` のようなパスが自動で入力されます。そのまま Return キーを押してください。

3. 続けて以下を実行します

> ⚠️ **`npm install` は初回だけ！** 2回目以降は実行しなくて OK です。

```bash
# 初回だけ実行（少し時間がかかります）
npm install
```

```bash
# 毎回これで起動
npm run editor
```

4. ブラウザが自動で `http://localhost:3001` を開きます

> 開かない場合は Chrome で `http://localhost:3001` を手動で開いてください。

---

## 🎯 まず「動いた！」を確認する（初回必須）

いきなり本番データを触らず、**サンプルCSV で動作確認**してください。

1. 「📂 CSVを開く」をクリック
2. `sample/beginner-sample.csv` を選ぶ
3. 3行のテロップが表示されたら成功
4. 「こんにちは！」の文字を変えて「⚡️ 保存して更新」を押す
5. 変化が確認できたら本番データへ進んでください

---

## 🟩 Step 2：CSV を読み込む

### サーバーモードで起動している場合
画面上部の **「⚡ サーバーモード（自動保存）」** ボタンが緑になっていれば、
CSVは自動で読み込まれます。

### ファイルから読み込む場合
1. **「📂 CSVを開く」** ボタンをクリック
2. `input/subtitles.csv` を選択
3. テロップ一覧が表示されます

---

## 🟨 Step 3：テロップを編集する

各テロップ行には以下の項目があります：

| 項目 | 説明 | 例 |
|------|------|-----|
| 開始 s | テロップが出始める秒数 | `4.92` |
| 終了 s | テロップが消える秒数 | `7.6` |
| テロップ文字 | 画面に表示する文字 | `成功する人って全部同じことしてる` |
| サイズ | 文字の大きさ（数字が大きいほど大） | `68` |
| 文字色 | 文字の色（カラーピッカーか16進数で指定） | `#ffffff`（白） |
| 縁取り色 | 文字の縁の色 | `#000000`（黒） |
| 背景色 | テロップの背景色（「なし」でOFF） | `#003366` |
| 効果音 | テロップが出るときに鳴る音 | `se_fanfare` |
| 縦位置 | 下から何ピクセルの位置に出すか | `360`（中央より下） |
| プレビュー | 見た目の確認（自動で更新） | — |

### 縦位置の目安

| 数値 | 位置 |
|------|------|
| 100 | 画面の一番下近く |
| 360 | 下から1/3あたり（デフォルト） |
| 700 | 画面の真ん中あたり |
| 1200 | 画面の上の方 |

## 🟪 Step 4：保存して動画に反映する

### ボタンの使い分け

| ボタン | いつ使う |
|--------|---------|
| 💾 保存（自動書き込み） | CSV だけ保存したいとき |
| ⚡️ 保存して更新 | **動画に反映させたいとき（メインで使う）**<br>ショートカット：キーボード左下の `⌘ Command` + `Return` |

### 手順

1. **「⚡️ 保存して更新」** を押す（または `Cmd + Return`）
2. 「✅ transcript.json を更新しました」と出たら成功
3. Remotion Studio のブラウザタブで **`Cmd + R`** を押してリロード
4. タイムラインで確認

---

## 🗑️ テロップを削除する

各行の左端にある **赤い「✕」ボタン** を押すと削除できます。

> ⚠️ 確認なしに削除されます。間違えた場合はブラウザをリロードしてやり直してください。

---

## ➕ テロップを追加する

画面下部の **「＋ テロップを追加」** ボタンで新しい行を追加できます。

---

## 🔁 スタイルをリセットする

**「スタイルをリセット」** ボタンを押すと、すべてのテロップのスタイル（サイズ・色）をデフォルトに戻します。

> ⚠️ 全行が対象です。特定の行だけリセットすることはできません。
