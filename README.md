# 🎬 テロップエディター / Telop Editor

**初心者でも使えるAI動画用テロップ編集ツール**

✅ 初心者OK　✅ 5分で起動　✅ コピペだけで動作　✅ 音声波形でタイミング調整

![テロップエディター画面](docs/screenshots/editor-screenshot.png)

音声ファイルと台本JSONを `input/` フォルダに入れるだけで、波形を見ながらテロップのタイミングをドラッグ調整できるツールです。

---

## ✨ できること

### 🎵 波形エディター（NEW）
- **音声波形を見ながらテロップをドラッグ**してタイミングを直感的に調整
- ブロックの端をドラッグして開始・終了時間を個別に変更
- Space キーで再生・停止、← → キーで1フレーム単位の微調整
- テロップ一括シフト（全体を早める・遅らせる）
- 選択した行だけまとめてシフト

### 📝 テロップ編集
- テロップの**文字・開始秒・終了秒**を自由に編集
- **フォントサイズ・文字色・縁取り色・背景色**を変更
- **効果音**を各テロップに紐付け
- **縦位置（画面のどこに表示するか）**を調整
- 編集結果を1クリックで動画生成データに反映（`⚡️ 保存して更新`）
- リアルタイム**プレビュー**付き

### 🤖 Whisper 自動タイムスタンプ（任意）
- AI音声認識（Whisper）で音声から自動的にテロップのタイミングを生成
- 手動で時間を入力する必要がなくなる

## 🚀 はじめかた（初めての方はここから）

### Step 1：Node.js をインストール

1. [https://nodejs.org/ja/](https://nodejs.org/ja/) を開く
2. **「LTS（推奨版）」** と書かれたボタンをクリックしてダウンロード
3. ダウンロードしたファイルをダブルクリックして「続ける」を押し続けてインストール完了

インストールできたか確認する方法：

```bash
node -v
```

`v18.x.x` 以上（`v20.x.x` 推奨）が表示されれば OK です。

---

### Step 2：ZIP を解凍してフォルダを開く

1. ダウンロードした ZIP ファイルをダブルクリックして解凍
2. 解凍されたフォルダを**デスクトップや書類フォルダ**に移動

---

### Step 3：ターミナルを起動してアプリを立ち上げる

1. Mac の **「Launchpad（ロケット型アイコン）」** を開く
2. 検索欄に「ターミナル」と入力して開く
3. ターミナルが開いたら、**解凍したフォルダをターミナルの黒い画面にドラッグ＆ドロップ**

> 💡 ドラッグすると `cd /Users/あなたの名前/...` のようなパスが自動で入力されます。そのまま Return キーを押してください。

4. 続けて以下を実行します

> ⚠️ **`npm install` は初回だけ実行してください。2回目以降は不要です。**

```bash
# ↓ 初回だけ実行（少し時間がかかります）
npm install
```

```bash
# ↓ 毎回これで起動します
npm run editor
```

5. ブラウザが自動で開き、`http://localhost:3001` でテロップエディターが表示されます

> ブラウザが開かない場合は、Chrome で `http://localhost:3001` を手動で開いてください。

> ⚠️ ターミナルを閉じるとアプリも止まります。使用中は開いたままにしてください。

## 🎯 基本の使い方（2ステップ）

### Step A：ファイルを準備する

`input/audio/` フォルダに音声ファイル（`.mp3` または `.wav`）を入れます。

```
auto-short-video-maker/
└── input/
    └── audio/
        └── あなたの音声.mp3  ← ここに入れる
```

テロップのタイミングは以下のいずれかで準備します：

**① JSONファイルを使う場合**（おすすめ）  
台本JSONを `input/` フォルダに入れて `npm run editor` を起動するだけ。自動で読み込まれます。

```json
{
  "title": "動画タイトル",
  "duration": 122,
  "audio": "sounds/audio.mp3",
  "segments": [
    { "id": 0, "start": 0.0, "end": 3.0, "text": "最初のテロップ" },
    { "id": 1, "start": 3.0, "end": 6.0, "text": "次のテロップ" }
  ]
}
```

**② Whisper で自動生成する場合**  
音声から自動でタイムスタンプを作成します（要 openai-whisper インストール）：

```bash
python3 scripts/generate-transcript.py input/audio/あなたの音声.mp3
```

### Step B：波形エディターで調整する

エディターを起動すると音声波形とテロップブロックが表示されます。

| 操作 | 動作 |
|------|------|
| ブロックをドラッグ | テロップ全体を移動 |
| ブロックの右端をドラッグ | 終了時間を調整 |
| ブロックの左端をドラッグ | 開始時間を調整 |
| `Space` キー | 再生・停止 |
| `←` キー | 1フレーム進む |
| `→` キー | 1フレーム戻る |
| ⏪ 全体を早める | 全テロップを一括で前にずらす |
| 全体を遅らせる ⏩ | 全テロップを一括で後ろにずらす |

調整が終わったら「**⚡️ 保存して更新**」を押せば完了です。

## 📝 ボタン一覧

| ボタン | 説明 |
|--------|------|
| 📂 CSVを開く | CSV ファイルを読み込む |
| 📄 JSONを読み込む | テロップ JSON ファイルを読み込む |

| 📂 音声を開く | 音声ファイルを手動で選択して波形を表示 |

> 💡 **ショートカット：`⌘ Command` + `Return` で「保存して更新」が実行されます。**

## ❓ よくある質問

**Q. ブラウザが自動で開かない**  
→ Chrome で `http://localhost:3001` を手動で開いてください。

**Q. 音声波形が表示されない**  
→ `input/audio/` フォルダに音声ファイルが入っているか確認してください。または「📂 音声を開く」から手動で選択してください。

**Q. `npm install` でエラーが出る**  
→ Node.js が正しくインストールされているか確認してください（`node -v` を実行）

**Q. 「保存して更新」でエラーが出る**  
→ ターミナルで `Ctrl + C` を押してから `npm run editor` を再実行してください。その後ブラウザを `Cmd + Shift + R` でリロードしてください。

**Q. Whisper のタイムスタンプがズレる**  
→ 波形エディターでドラッグして微調整してください。または「⏪ 全体を早める / 遅らせる ⏩」で一括調整できます。

詳しいトラブル対応 → [FAQ.md](docs/FAQ.md) を見てください。
