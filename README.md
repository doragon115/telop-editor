# 🎬 平松悟のショート動画メーカー

**Mac初心者でも、最後まで完成できる**ショート動画制作ツールです。

✅ プログラミング知識ゼロでOK　✅ コピペだけで動く　✅ 壊れない設計　✅ 完走サポート付き

---

## 📌 まず最初にここを読んでください

「ダウンロードしたけど何をすればいい？」という方は、この順番で進めてください。

```
① Node.js をインストールする（初回だけ・5分）
② ZIPを解凍してフォルダをデスクトップに置く
③ ターミナルに1行打ち込んで起動する
④ ブラウザでエディターが開く
⑤ 音声を入れてテロップを調整する
⑥ 動画を書き出す
```

難しそうに見えますが、一つひとつは「次にここを押す」だけです。  
迷ったら下の手順を読んでください。

## 📦 ダウンロード方法

1. このページ上部の緑の「**Code**」ボタンをクリック
2. 「**Download ZIP**」を選択
3. ダウンロードされたZIPファイルを**ダブルクリックして解凍**
4. 解凍されたフォルダを**デスクトップに移動**

---

## 🚀 初回セットアップ（初めての方はここから）

### STEP 1：Node.js をインストールする

> ⚠️ **「Node.jsって何？」という方へ**  
> アプリを動かすために必要なエンジンです。インストールするだけでOK、使い方は覚えなくて大丈夫です。

1. [https://nodejs.org/ja/](https://nodejs.org/ja/) を開く
2. 「**LTS（推奨版）**」と書かれた緑のボタンをクリック
3. ダウンロードしたファイルをダブルクリック → 「続ける」を押し続ける
4. インストール完了

---

### STEP 2：ターミナルを起動する

「ターミナル」とは黒い画面に文字を打ち込むアプリです。怖くないです。1行だけ入力します。

1. キーボードの **「⌘（コマンド）＋スペース」** を押す
2. 「**ターミナル**」と入力してEnterを押す
3. 黒い画面が開いたら準備完了

---

### STEP 3：フォルダをターミナルに放り込む

1. デスクトップに置いた解凍済みフォルダ（`telop-editor-main` など）を  
　 ターミナルの画面に**ドラッグ＆ドロップ**する
2. `cd /Users/あなたの名前/Desktop/telop-editor-main` のような文字が入力される
3. **Enterキー**を押す

---

### STEP 4：初回のみ「準備コマンド」を実行する

> ⚠️ **この操作は初回だけです。2回目以降は不要です。**

ターミナルに以下を**コピペ**して、Enterを押してください。

```
npm install
```

しばらく文字がたくさん流れます（2〜5分かかる場合あり）。  
止まったら完了です。

---

### STEP 5：エディターを起動する

```
npm run editor
```

ブラウザが自動で開いて、エディター画面が表示されます。  
開かない場合は、Chromeで `http://localhost:3001` を手動で開いてください。

> 💡 **ターミナルは開いたまま**にしてください。閉じるとアプリも止まります。

---

## 🎯 動画を作る手順

### ① 音声ファイルを入れる

フォルダの中の **`input/audio/`** フォルダに、音声ファイル（`.mp3` または `.wav`）を入れてください。

```
telop-editor-main/
└── input/
    └── audio/
        └── ここに音声ファイルを入れる.mp3
```

---

### ② エディターで調整する

エディターが開いたら、音声波形とテロップブロックが表示されます。

| 操作 | 動作 |
|------|------|
| ブロックをドラッグ | テロップの位置を移動 |
| ブロックの端をドラッグ | 開始・終了時間を調整 |
| **Space** キー | 再生・停止 |
| **⌘ + Enter** | 保存して動画データを更新 |

---

### ③ 動画を書き出す

エディター右上の「**🎬 動画を書き出す**」ボタンを押せば完成です。  
`output/` フォルダに動画ファイルが保存されます。

---

## 🛠️ エラーが出たときは

**「壊したかも」と思っても大丈夫です。自動バックアップがあります。**

よくあるエラーと対処法：

| こんな表示が出たら | やること |
|----------------|---------|
| ブラウザが開かない | Chromeで `http://localhost:3001` を手動で開く |
| 音声波形が表示されない | `input/audio/` に音声ファイルが入っているか確認 |
| `npm install` でエラー | `node -v` を実行してNode.jsが入っているか確認 |
| 画面が固まった | ターミナルで `Ctrl+C` → `npm run editor` を再実行 |
| 動画が書き出せない | ターミナルのエラー文を見て、わからなければ問い合わせへ |

## 🎵 BGM・効果音について

`public/sounds/` フォルダに音声ファイルが入っています。  
エディターのドロップダウンから選ぶだけで設定できます。

---

## 🖼️ 顔の位置・挿入画像について

- **顔の位置**：エディター上部の「👤 顔の位置」で左・中央・右を選択できます
- **挿入画像**：`input/images/inserts/` に画像を入れると動画内に表示できます
- **画像分割ツール**：人物写真を4×4の16枚に分割する専用ツールが付属しています（`image-splitter.html`）

## ❓ よくある質問

**Q. Macのパスワードを求められた**  
→ Node.jsインストール時に求められることがあります。Macのログインパスワードを入力してください。

**Q. 途中でやめたい**  
→ 今の状態で保存されています。次回また `npm run editor` で再開できます。

**Q. フォルダを移動してしまった**  
→ STEP 3をもう一度やり直して、新しい場所からフォルダをターミナルにドロップしてください。

**Q. 英語がたくさん出てきて怖い**  
→ ターミナルに英語が流れるのは正常です。最後に `$ ` が表示されれば完了のサインです。
